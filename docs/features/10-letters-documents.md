# Feature 10 — Letters & Documents

## Overview

The Letters screen automates generation, customisation, delivery, and signing of all standard HR correspondence. Atlas AI recommends the right template based on employee status and auto-populates all variables from the HR database.

**Screen ID:** `letters`  
**File:** [screens/letters.jsx](../../screens/letters.jsx)  
**Portal:** Admin only

---

## 10.2 Template Library

Eight standard HR letter templates:

| Template | Purpose |
|---|---|
| Salary Certificate | Formal salary confirmation for banks, visa applications |
| Offer Letter | Employment offer with compensation and joining terms |
| Experience Letter | Service confirmation for departing employees |
| Increment Letter | Salary revision notification with old and new CTC |
| Appointment Letter | Formal joining confirmation for new hires |
| Relieving Letter | Final-day confirmation for resigned employees |
| Promotion Letter | Role and grade change notification |
| Warning Letter | Performance or conduct advisory |

Each template card shows: icon, category chip, template name, number of uses, last edited date, variable count.

---

## 10.3 Employee Selection & Atlas Recommendation

- **Searchable employee dropdown** — avatar, name, ID, designation, department, status badge
- When an employee is selected, **Atlas analyses their current status** and recommends the most relevant template
  - Example: recommends "Increment Letter" if a recent CTC revision is detected
- Recommendation pills are colour-coded chips with a reason explaining why Atlas recommends each

---

## 10.4 Variable Editor

Left panel for customising letter variables.

- Each variable shown as `{{variable_name}}` syntax key with an editable value field
- **Auto-filled indicator** (green checkmark) — Atlas has pre-populated the value from the HR database
- Manually editable for custom or situational values

---

## 10.5 Delivery Configuration

Toggles for how the letter will be delivered:

| Channel | Description |
|---|---|
| PDF Download | Saves locally |
| Email delivery | Sends to employee's registered email |
| Self-Service Portal | Makes available in employee's document library |
| WhatsApp link | Generates a secure share link |
| DocuSign E-Sign | Routes for digital signature workflow |

---

## 10.6 Document Preview

Right panel showing a pixel-accurate document preview — updates in real-time as variables are edited.

**Document includes:**
- Company letterhead (logo, name, address, reference number, date)
- Fully populated letter body with all resolved variables
- Signature block (authorised signatory name and title)
- Footer with verification URL and document hash for authenticity

---

## 10.7 Recent Letters Log

Table of recently generated letters.

**Columns:** Letter ID, Template used, Employee, Delivery channel, Generation date, Status, Download

- Auto-generated badge distinguishes Atlas-created letters from manually generated ones
