# Feature 08 — Payslips

## Overview

The Payslips screen handles generation, preview, and distribution of employee payslips. It provides a pixel-accurate, print-ready payslip document with PDF download, email delivery, and a bulk-generation summary dashboard.

**Screen ID:** `payslips`  
**File:** [screens/payslips.jsx](../../screens/payslips.jsx)  
**Portal:** Admin only

---

## 8.1 Distribution Summary

Four KPI cards showing payslip health for the current period:

| Card | Example | Description |
|---|---|---|
| Generated | 100% | Percentage of payslips generated |
| E-Delivered | 97% | Percentage successfully sent via email or portal |
| Opened | 88% | Percentage of delivered payslips viewed by the employee |
| Disputes Raised | 3 (2 resolved) | Payslip disputes raised this period |

---

## 8.2 Employee Selector

Left-side panel for browsing and selecting employees.

- **Search box** — filter by name or ID
- **Period dropdown** — switch between payroll months
- **Scrollable list:** avatar, full name, ID, department, monthly net salary
- Selected employee highlighted with left border accent + background tint

---

## 8.3 Payslip Preview

Main content area showing a pixel-accurate, print-ready payslip document.

### Document Actions

| Button | Action |
|---|---|
| Preview PDF | Renders payslip in a print-preview modal |
| Download | Saves as PDF file |
| E-mail | Sends to employee's registered email address |

### Document Structure

**Company Header**
- Source One logo badge, company name, full address, TAN, CIN

**Payslip Metadata**
- Month/year period, generation date

**Employee Information Grid (2 columns)**
- Name, Employee ID, Designation, Department, Location, Date of Joining
- PAN, Bank Account (masked), UAN, Days Worked

**Earnings Table**

| Component | Description |
|---|---|
| Basic | Base salary |
| HRA | House Rent Allowance |
| Special Allowance | Variable component |
| LTA | Leave Travel Allowance |
| Telephone Allowance | Communication allowance |
| **Total Earnings** | Sum of all earnings |

**Deductions Table**

| Component | Rate |
|---|---|
| Provident Fund | 12% of basic salary |
| Professional Tax | State-specific (Karnataka rate) |
| TDS | As per IT declaration |
| **Total Deductions** | Sum of all deductions |

**Net Pay Highlight Box**
- Large rupee amount (e.g., ₹42,380)
- Amount in words ("Rupees Forty-Two Thousand Three Hundred Eighty only")

**Footer**
- Payslip authenticity disclaimer
- "Generated by Atlas" attribution
- Cryptographic hash for integrity verification
